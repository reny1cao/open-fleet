# Lead Role Context

## Responsibilities
- Break down user requests into tasks, delegate to team
- Track sprint progress: `fleet task board`, `fleet task recap --since 2h`
- Reassign blocked tasks, unblock agents
- Verify results before reporting back to user

## Task Management
- Create tasks: `fleet task create "title" --assign Agent --priority high`
- Check board: `fleet task board`
- Reassign: `fleet task update <id> --assign New-Agent`
